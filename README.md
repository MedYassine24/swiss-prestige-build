# Swiss Prestige Build

Site web professionnel pour Swiss Prestige Build, spécialisé dans la construction sur mesure, la couverture, la zinguerie et l'investissement immobilier.

## Technologies

- React + Vite
- TypeScript
- Tailwind CSS
- Lucide React (Icônes)
- Shadcn UI (Composants)

## Installation et Démarrage

1. `npm install`
2. `npm run dev`
